# 🛍️ Chinnu Store - E-commerce Website

Chinnu Store is a simple front-end E-commerce website developed using HTML, CSS, and JavaScript.  
It simulates an online shopping experience with cart management, order history, and a basic payment system.

---

## ✨ Features

- 🛒 Add to Cart functionality
- 💳 Simulated Payment System
- 📦 Order History (stored using localStorage)
- ⭐ Customer Reviews Section
- 🏪 About Store Page
- 📱 Simple and clean UI design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## 💡 How It Works

- Users can browse products and add them to the cart
- Cart automatically calculates total price
- On clicking "Pay", a fake payment process is triggered
- Orders are saved in browser localStorage
- Users can view previous orders anytime
- Reviews are also stored using localStorage

---

## ⚠️ Note

This is a front-end project only.  
The payment system is not real and is implemented only for demonstration purposes.
